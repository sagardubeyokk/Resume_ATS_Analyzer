# Resumer — AI Resume Analyzer

An intelligent resume analysis tool built with Streamlit and powered by LLaMA 3 via Groq API.

## Features
- Upload any resume in PDF format
- Enter your target job role
- Get instant AI-powered feedback including:
  - ATS Score (out of 100)
  - Key Strengths
  - Weaknesses
  - Missing Keywords
  - Top 5 Improvement Suggestions

## Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3)
- pdfplumber
- python-dotenv

## Setup
1. Clone the repo
2. Install dependencies: pip install -r requirements.txt
3. Add your Groq API key in .env file
4. Run: streamlit run app.py
